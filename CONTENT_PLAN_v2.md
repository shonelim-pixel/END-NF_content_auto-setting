# END NF 콘텐츠 시스템 v2 — 전체 재설계 계획

> 작성일: 2026-03-07
> 핵심 철학: **카페를 찾는 환자와 가족에게 진짜 힘이 되는 콘텐츠**

---

## 우리가 해결하려는 문제

카페를 찾는 사람들은 크게 두 가지 순간에 온다:

**1. 처음 — 아이에게 의심이 들 때, 진단을 받았을 때**
> 모든 게 내 탓 같고, 세상이 무너지는 느낌.
> "우리 아이가 앞으로 어떻게 되는 거지?" 하는 공포.
> 인터넷 검색이 더 무섭게 만들고, 혼자라는 느낌.

**2. 중간에 — 끝나지 않는 고통으로 희망이 사라질 때**
> 치료도 없고, 나아지지도 않고, 끝이 안 보이는 느낌.
> 주변은 이해 못 하고, 지쳐가는 마음.
> "이게 언제까지 계속되는 거지?"

이 두 순간에 카페에서 만나는 글이 **"당신만 그런 게 아니에요"**, **"우리도 그랬지만 지금은 이렇게 지내고 있어요"**, **"새로운 치료법이 다가오고 있어요"**라는 메시지를 전할 수 있어야 한다.

## 2. 1회차: "💛 우리의 이야기" (화요일)

### 콘텐츠 철학
단순히 "감동적인 이야기"가 아니라, **지금 힘든 사람이 읽고 조금이라도 마음이 나아지는 글**.
"나만 그런 게 아니구나", "다른 가족도 이렇게 지내고 있구나", "아이가 행복할 수 있구나"를 느끼게 해주는 것이 목표.

### 콘텐츠 유형 (6가지)

#### A. 💬 "처음엔 나도 그랬어요" — 선배 가족의 진단 이야기
> **대상 독자: 막 진단받은 가족**

- 진단 직후의 충격, 죄책감, 눈물의 시간을 솔직하게
- "그때 이런 게 도움이 됐어요" 실제 경험
- 시간이 지나면서 마음이 어떻게 변했는지
- **핵심 톤**: 영웅담이 아닌 현실적 이야기. "지금도 가끔 힘들지만, 이렇게 지내고 있어요"
- **소스**: CTF Patient Advocates 인터뷰, Reddit 진단 후기, 레어노트 환자 인터뷰
- **연구 근거**: 진단 전달 방식이 가족의 장기 심리에 큰 영향 (OAEP, 2022)

#### B. 😊 "아이가 웃는 순간들" — 질환이 전부가 아닌 일상
> **대상 독자: 아이의 미래가 걱정되는 부모**

- NF 아이가 학교에서, 친구와, 가족과 행복한 순간들
- 좋아하는 것을 찾은 이야기, 작은 성취
- CTF National Ambassador(2026: Alexandra Norton) 같은 롤모델
- **핵심 메시지**: "질환은 아이의 일부이지, 전부가 아닙니다"
- **소스**: CTF 인스타그램, NF Hero 시리즈, 해외 NF 유튜버/크리에이터

#### C. 🤝 "함께라서 괜찮아요" — 커뮤니티가 주는 힘
> **대상 독자: 고립감을 느끼는 환자/가족**

- 혼자 고민하다 환우회를 찾은 후 달라진 이야기
- 온/오프라인 모임에서 "나만 그런 게 아니었구나"를 느낀 순간
- 해외 NF 커뮤니티(CTF, NF Network) 활동 소개
- **핵심 메시지**: "이 글을 읽고 있는 당신도 이미 혼자가 아닙니다"
- **소스**: CTF Cupid's Undie Run, NF Walk, 국내 END NF 활동, Reddit 커뮤니티 사연

#### D. 🧘 "마음이 힘들 때" — 실제로 도움이 된 마음 챙김 방법
> **대상 독자: 끝이 안 보여서 지친 환자/가족**

- 환자·가족이 직접 찾은 대처법 (전문가 조언이 아닌 실제 경험)
- "검색을 멈추기로 한 날", "같은 처지의 부모 모임에서 처음 울었던 날"
- 가족 내 소통 방법, 부부가 함께 버텨온 이야기
- 간병 피로, 형제자매 소외감 등 가족 전체의 마음 돌봄
- **연구 근거**: 가족 내 소통과 긍정적 태도가 대처에 핵심 (Springer, 2024)
- **소스**: Reddit 경험담, CTF Mental Health Resources, Global Genes 심리 지원 가이드

#### E. 🎬 영화·책·미디어 속 이야기
> **대상 독자: 모든 환자/가족**

- 희귀질환이나 "다름"을 다룬 영화, 책, 다큐멘터리
- 영화 '원더', 희귀질환 관련 최근 한국/해외 영화
- Rare Disease Day 영상, NF 다큐멘터리
- **핵심**: 단순 리뷰가 아니라 "NF 가족으로서 이 장면에서 어떤 감정을 느꼈는지" 공감 포인트

#### F. 🌟 해외 커뮤니티·이벤트 소식
> **대상 독자: 글로벌 연대감을 느끼고 싶은 분**

- CTF 이벤트, 캠페인, 기금 모금 소식
- NF 인식의 달(5월), Rare Disease Day(2월)
- 해외 NF 클리닉 네트워크 확장 (현재 약 80개소)
- CTF National Gala, 해외 환우회 활동
- **핵심 메시지**: "전 세계가 함께 싸우고 있습니다"

### 소재 선택 로직 (자동)
```
1. 최근 7일 이내 신규 소재 검색 (CTF, Reddit, 레어노트, 뉴스)
2. 신규 있음 → 유형 A~F 중 가장 적합한 것으로 분류 → 콘텐츠 생성
   - 나머지 미사용 소재는 backlog(저장소)에 보관
3. 신규 없음 → backlog에서 미사용 소재 1건 꺼내기
4. backlog도 비었음 → 에버그린 콘텐츠 (유형 D, E 중 미사용 건)
5. 생성 후 sent_history에 기록 (60일간 재사용 금지)
```

### 에버그린 콘텐츠 목록 (폴백용, 소재 없을 때 사용)
```
eg-001: "NF 진단을 받은 날, 그리고 1년 후" (유형 A)
eg-002: "검색을 멈추고 아이를 안아주세요" (유형 D)
eg-003: "학교에서 아이를 위해 할 수 있는 것들" (유형 B)
eg-004: "간병하는 가족의 마음도 소중합니다" (유형 D)
eg-005: "NF 아이와 함께하는 특별한 일상" (유형 B)
eg-006: "혼자 울던 밤에서, 함께 웃는 오늘까지" (유형 C)
eg-007: "영화 '원더'가 NF 가족에게 전하는 메시지" (유형 E)
eg-008: "부부가 함께 NF를 이겨내는 법" (유형 D)
eg-009: "형제자매에게 NF를 설명하는 방법" (유형 B)
eg-010: "마음이 무너질 때 도움받을 수 있는 곳" (유형 D)
```

## 3. 2회차: "💡 알아두면 좋은 소식" (금요일) — 월별 로테이션

### 로테이션 구조

| 주차 | 서브 주제 | 주요 소스 |
|------|-----------|-----------|
| **1주차** | 🔬 NF 연구·신약·임상시험 | PubMed, ClinicalTrials.gov, CTF Drug Pipeline |
| **2주차** | 🧬 유전질환 치료 연구 동향 | PubMed (gene therapy, CRISPR, precision medicine) |
| **3주차** | 🌍 CTF·글로벌 NF 소식 | CTF News/인스타, NF Network, NORD |
| **4주차** | 🔬 또는 🧬 (그 달에 더 새로운 쪽) | 자동 판단 |

> 5주차가 있는 달: "이달의 NF 소식 총정리" 또는 CTF 특집

### 콘텐츠 가이드

#### 1주차: NF 연구·신약·임상시험
- NF 관련 최신 논문 (최근 30일 검색)
- 임상시험 현황 (신규 등록, 결과 발표, 🆕 하이라이트)
- CTF-Healx AI 신약 발견 프로젝트
- Koselugo 유럽 승인 등 글로벌 치료제 동향
- **새 소식 없을 때 → 교육 시리즈**:
  - "MEK 억제제, 쉽게 알려드릴게요"
  - "임상시험은 어떤 단계를 거치나요?"
  - "AI가 신약을 찾는다고요? CTF-Healx 이야기"
  - "신약이 환자에게 오기까지의 여정"
- **핵심 톤**: 과도한 기대 없이 현실적이되, 희망을 잃지 않게

#### 2주차: 유전질환 치료 연구 동향 (장기 확장 방향)
- CRISPR/유전자 편집 치료 최신 동향
- 유전자 치료(Gene Therapy) 글로벌 임상 진전
- 정밀의료(Precision Medicine) 발전
- RAS 경로 표적 치료 (NF1과 직접 관련)
- **NF와의 연결고리 필수**: "이 기술이 NF 치료에도 희망이 됩니다"
- **핵심 톤**: NF → 유전질환으로의 자연스러운 확장. 너무 어렵지 않게.
- PubMed 추가 검색어:
  - "gene therapy rare disease"
  - "CRISPR genetic disorder treatment"
  - "precision medicine inherited disease"
  - "RAS pathway targeted therapy"

#### 3주차: CTF·글로벌 NF 소식
- CTF 공식 뉴스 (ctf.org/news)
- CTF 인스타그램 주요 포스트
- NF Research 펀딩 (미 의회 FY2026 $25M NF 연구비)
- NF 클리닉 네트워크 확장 (80개소)
- 해외 단체 소식 (NF Network Europe, CTF Australia)
- **새 소식 없을 때**: "CTF가 걸어온 길", "글로벌 NF 커뮤니티 지도"

#### 4주차: 교차 주제
- 1·2주차 중 그 달에 더 새로운 소식이 있는 쪽으로 자동 선택
- 또는 해당 월의 특별 이슈 (NF 인식의 달 5월, Rare Disease Day 2월 등)

### 2회차 반복 방지
```
1. 최근 30일 데이터 수집 (7일보다 넓게)
2. sent_history와 대조하여 이미 다룬 논문/임상시험 제외
3. 신규 소재 있으면 → 사용
4. 없으면 → 교육 시리즈 중 미사용 건 선택
5. 교육 시리즈도 다 썼으면 → "현재까지의 NF 치료 현황 총정리" 형식
```

## 4. 특별편: "📋 복지 가이드 시리즈"

> 정기 발송과 별도로, **월 1~2회** 특별편으로 발행
> NF뿐 아니라 희귀질환·의료 복지 전반을 **매우 상세하고 실용적으로** 안내
> 한 주제를 깊이 있게, 환자/가족이 바로 활용할 수 있도록

### 시리즈 목록 (2026년 기준)

| 회차 | 주제 | 핵심 내용 |
|------|------|-----------|
| 1 | **산정특례 신청 완전 가이드** | 대상 질환(2026년 1,389개), 신청 절차, 필요 서류, 5년 재등록 간소화 |
| 2 | **2026년 달라지는 의료 복지 총정리** | 본인부담률 10%→5% 인하, 치료제 급여 등재 240→100일 단축, 부양의무자 기준 단계적 폐지 |
| 3 | **희귀질환 의료비 지원사업** | 본인부담금 지원, 간병비, 특수식이 비용, 소득 기준, 신청 방법 |
| 4 | **장애등록과 장애인 복지** | 장애등록 절차, 장애등급 판정, 의료비 감면, 보조기기 지원 |
| 5 | **아이를 위한 교육·돌봄 지원** | 특수교육 대상자 지정, 장애아 돌봄 서비스, 보육료 지원, 발달재활 바우처 |
| 6 | **긴급복지 지원제도** | 위기상황 시 긴급 의료비·생활비 지원, 신청 방법 |
| 7 | **유전 상담과 검사 안내** | 유전 상담이란, 어디서 받나, 비용, NF 유전 검사 종류 |
| 8 | **간병·돌봄 가족을 위한 지원** | 가족 돌봄 휴가, 간병인 지원, 심리 상담 서비스, 가족 쉼터 |
| 9 | **건강보험·실비보험 청구 가이드** | 실비보험 청구 방법, 산정특례 환자 보험 활용, 비급여 항목 |
| 10 | **해외 치료를 위한 지원제도** | 해외 치료비 지원, 건강보험 적용, 출국 전 준비 |
| 11 | **2026 건강보험 종합계획 핵심 정리** | 건강보험료율 7.19%, 의료-복지 연계 포괄 지원체계 |
| 12 | **마음 건강을 위한 지원 — 환자와 가족 심리 상담** | 희귀질환 가족 심리 상담 서비스, 정신건강복지센터, 자조모임 안내 |

### 발행 방식
- 월 1~2회, 정기 발송일(화 또는 금)에 **정기 콘텐츠 대신** 발행
- 또는 정기 콘텐츠와 **별도 추가 발행** (workflow_dispatch로 수동 트리거)
- 별도 해시태그: #ENDNF복지가이드 #희귀질환복지 #의료비지원
- 카페 내 "복지 정보" 게시판에 아카이빙

## 5. 심리 힐링 요소 — 모든 콘텐츠에 녹이기

### 원칙
모든 글(1회차, 2회차, 특별편)에 아래 요소 중 최소 1개를 자연스럽게 포함:

| 요소 | 설명 | 예시 |
|------|------|------|
| **"혼자가 아닙니다"** | 고립감 해소 | "전 세계 250만 NF 환자가 함께하고 있습니다" |
| **"걱정보다 사랑을"** | 아이 관련 불안 해소 | "아이에게 가장 필요한 건 치료 정보보다 엄마 아빠의 웃는 얼굴입니다" |
| **"한 걸음씩"** | 끝이 안 보이는 느낌 해소 | "10년 전에는 치료제가 0개였지만, 지금은 2개가 승인되었습니다" |
| **"당신의 마음도 소중합니다"** | 보호자 번아웃 해소 | "아이를 돌보는 만큼 자신의 마음도 돌봐주세요" |
| **"이해합니다"** | 공감과 인정 | "처음 진단받았을 때의 그 마음, 우리도 알고 있습니다" |

### 진단 초기 가족을 위한 특별 배려
- 1회차 콘텐츠에 항상 마무리 부분에 "처음 진단받으셨나요?" 안내 포함
- END NF 카페의 "진단 초기 가이드" 게시글 링크
- "혼자 고민하지 마세요, 카페에 질문을 남겨주시면 선배 가족이 답해드립니다" 문구

## 7. 반복 방지 시스템

### content_history.json (신규 파일)
```json
{
  "sent_items": [
    {
      "date": "2026-03-03",
      "content_type": "story",
      "story_type": "A",
      "source_id": "ctf_post_123",

    }
  ],
  "backlog": [
    {
      "collected_date": "2026-02-25",
      "source_id": "reddit_abc456",
      "source_type": "patient_story",
      "title": "진단 후 3년, 지금 우리 가족은",
      "story_type": "A",
      "used": false
    }
  ],
  "evergreen_index": {
    "eg-001": { "last_used": null },
    "eg-002": { "last_used": "2026-02-11" }
  },
  "education_series_index": {
    "edu-001": { "topic": "MEK 억제제란?", "last_used": null },
    "edu-002": { "topic": "임상시험 단계 이해하기", "last_used": null }
  },
  "rotation_log": {
    "2026-03": { "week1": "research", "week2": "gene_therapy", "week3": "ctf_news", "week4": "research" }
  }
}
```

### 전체 플로우
```
[화요일 - 우리의 이야기]
1. 최근 7일 데이터 수집 (CTF, Reddit, 레어노트, 뉴스)
2. sent_items와 대조 → 이미 사용한 소재 제외
3. 신규 소재 있으면:
   - 가장 적합한 1건 선택 → 유형 A~F 자동 분류 → 생성
   - 나머지는 backlog에 저장
4. 신규 없으면:
   - backlog에서 미사용 건 1개 꺼내기 (오래된 순)
5. backlog도 비었으면:
   - evergreen 중 가장 오래 미사용 건 선택
6. 생성 후 → sent_items 기록 + 텔레그램 발송

[금요일 - 알아두면 좋은 소식]
1. 이번 달 몇 주차인지 계산
2. 로테이션 주제 결정 (1~4주차)
3. 해당 주제에 맞는 데이터 수집 (30일 범위)
4. sent_items와 대조 → 이미 다룬 논문/임상시험 제외
5. 신규 소재 있으면 → 생성
6. 없으면 → 교육 시리즈 중 미사용 건 선택
7. 생성 후 → sent_items 기록 + 텔레그램 발송
```

## 9. 프롬프트 구조 (핵심)

### 1회차 프롬프트 — 심리 힐링 요소 내장

```
[시스템 프롬프트에 추가할 핵심 지침]

당신은 글을 쓸 때 항상 두 종류의 독자를 떠올려야 합니다:
1. 막 진단받아서 세상이 무너진 것 같은 부모
2. 오랜 투병으로 희망이 사라진 환자/가족

이 두 사람이 이 글을 읽었을 때 조금이라도 마음이 나아져야 합니다.

절대 하지 말 것:
- "힘내세요"만 반복하는 빈 위로
- 비현실적인 희망 (곧 완치됩니다 등)
- 성공 영웅담으로 부담 주기

반드시 할 것:
- "당신만 그런 게 아닙니다" 메시지
- 실제 경험에서 나온 구체적인 위로
- 작은 희망이라도 근거 있는 희망
- 아이 관련 이야기에는 "걱정보다 사랑과 행복을" 메시지
- 글 마지막에 "처음 진단받으셨나요? 혼자 고민하지 마세요" 안내
```
